 Defines a checkbox, which the user can toggle on or off.
type="color" Defines a color picker. The default value is #000000 (black).
type="date" Used for input fields that should contain a date.
type="email" Used for input fields that should contain an e-mail address.
type="file" Defines a file upload box with a browse button.
type="hidden" Defines a field within a form that is not visible to the user.
type="month" Allows the user to select a month and year.
type="number" Defines a numeric input field.
type="password" Displays an obfuscated password entry field.
type="radio" Female
Male

 Defines a circular selection button in a form.

type="reset" Defines a button on a form that will return all fields to their default values.
type="search" Used for search fields
type="submit" Defines a button that is clicked to submit a form.
type="tel" Used for input fields that should contain a telephone number.
type="text" Defines a text entry field in a form.
type="time" Allows the user to select a time (no time zone).
type="url" Used for input fields that should contain a URL address.
type="week" Allows the user to select a week and year.
